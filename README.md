# Components Documentation

## 1. Popup

A flexible popup menu component that supports:

- Custom dynamic positioning with target tracking
- Click and hover triggers
- Focus management

Features a data table with three different popup implementations (Menu, Popup click, Popup hover/click) and position controls for fine-tuning placement.

---

## 2. Tooltip

A tooltip component that extends the Popup component, featuring:

- Dynamic nested element targeting
- Fully decoupled two-way binding
- Rich content support

Includes examples of both one-way and two-way binding patterns, with interactive data tables demonstrating tooltip functionality with quantity sliders and text editing.

---

## 3. Guided Tour

A basic tour guide component using the Popup component.

Simple implementation for creating step-by-step guided tours through your application interface.

---

## 4. Ticked Slider

A slider component with clickable ticks and an input element over each handle.

Features multiple configurations including:

- Basic tick navigation
- Labeled ticks with value display
- Sticky input mode
- Range selection with dual handles
- Customizable tick spacing and decimal precision

---

## 5. Gauge

A knob component with a customizable pointer.

Displays circular gauge controls with adjustable:

- Size variations
- Pointer styles (height, width, offset)
- Value ranges and fulcrum positioning
- Optional center value display

---

## 6. Pivot Table

A component that dynamically groups tabular data by specifiable columns.

Features a product inventory table with:

- Multi-column grouping via dropdown selection
- Resizable columns
- Sortable headers
- Inline quantity editing
- Expandable/collapsible grouped rows

---

## 7. Tree Panel

A panel menu that works like the tree select.

Hierarchical navigation with:

- Expandable/collapsible sections
- Checkbox selection states (checked, unchecked, indeterminate)
- Badge counters for item counts
- Nested item support
- Multi-level hierarchy (Users, Tasks, Calendar sections)

---

## 8. Tree Table Filter

Custom filter functionality for the TreeTable with the following features:

- Match highlighting
- Auto unfold
- High performance

Includes global search and per-column filtering with real-time highlighting of matching text across hierarchical file system data.

---

## 9. Tree Table Misc.

Extending the TreeTable with the following functionality:

- Drag and drop
- Animated expansion
- Hierarchy lines

Visual enhancements showing parent-child relationships with connecting lines and smooth expand/collapse animations for tree nodes.

---

## 10. Date Completion

A directive that autocompletes dates while typing.

Provides intelligent date input with three modes:

- Date only (`dd/mm/yyyy`)
- DateTime (`dd/mm/yyyy hh:mm`)
- Time only (`hh:mm`)

Auto-completes as you type for faster data entry.
